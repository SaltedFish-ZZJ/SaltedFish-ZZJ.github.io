认知是一张漆黑的地图

从一个寓言 Prompt 开始，绕到提问能力评估，最后停在一个比喻上—— 它不精确，但足够好用。

一、0.92 系数

Anthropic 分析过 100 万条真实对话，得出一个数字：人类提问质量与 AI 回答质量的相关性系数 = 0.92。科学研究里 0.7 以上就算强相关，0.92 意味着——AI 几乎是提问者认知的放大器，近乎完美镜像。

大模型遇强则强，遇弱则弱。问题质量直接决定模型内部激活哪些"专家"。低质量的问题，很难激活高质量的答案。

二、我的提问模式自查

拿这个系数照了照自己，我的提问大致分三类：

  • A 类（操作型）——"怎么做""如何用"。占大多数。能用，但 AI 只能给说明书式的回答。
  • B 类（连接型）——"这个能不能跟 XX 结合"。偶尔出现，开始有自己的思考在里面。
  • C 类（深度型）——"要是没有这些杠杆，我又能干什么"。极少见。而且严格说，这类问题不是问 AI 的，是问自己的。

结论：提问能力不差，但没被有意识地开发。最大的短板是习惯性问"怎么做"，而不是"为什么"。

使用原则很简单：A 类日常省力，完全没问题；但当你需要 C 类答案时，必须问 C 类问题。不是每个问题都要 C 类，而是要知道什么时候该切换。

三、认知边界能探索吗

顺着这个想下去，我问了自己一个问题：认知边界到底能不能被探索？

我先想到的是：可能是伪命题——人在没有参照物的时候，根本不知道自己不知道什么。

然后我问了另一个模型，它给了相反的回答：是真命题——"不知道自己不知道什么"这个状态本身，就是被框在认知边界里的终极证明。它打了个比方：鱼在出水之前无法感知水，但它对世界的认知确实被水限制着。你不需要看到脑后，才知道视野有边界。

后来我想通了：这两个回答其实不矛盾。一个回答的是"认知边界是否存在"（存在），一个回答的是"认知边界能否被直接探索"（不能，只能碰撞）。

四、四种探测器，都有缺陷

方法原理适合场景
费曼检验说不清楚 = 没真懂你以为你懂的东西
连问五次为什么第三层卡住 = 只记了结论你信奉的观点
预测测试预测错了 = 认知有缺口你要做的决策
找矛盾两个信念打架 = 都没想透你的价值观

这四种方法有一个共同缺陷：全部靠自己。只能找到"知道自己不知道的"，找不到"不知道自己不知道的"。

所以必须补充外部输入：找观点完全不同的人聊天，去看高手怎么做同样的事——以及最准的一种：去实际做一个东西，看哪里卡住。行动是最高精度的探测器。

五、黑暗地图

聊到这里，一个比喻自己冒了出来：

认知是一张漆黑的地图。我学过的每一个知识、每一次思考，都是在点亮地图。尚未被点亮的地方，就是认知和思维的边界。

这个比喻有三层意思：

  • 地图是无限大的。没有人能学尽所有知识，选择点亮哪一块，比拼命多亮几块更重要。
  • 亮着的地方，亮度不一样。背了定义是微光；能解释给别人听是灯泡；实际用它做过决策是火炬；能教别人用它做决策，才算太阳。
  • 知识和认知不是一回事。知道一个事实，和理解它为什么重要、怎么用，是两回事。

行动启示：不相关的区域不用去亮。把跟自己有关的那几块从微光变成火炬——如无必要，勿增实体。

这篇文章的轨迹：寓言 Prompt → 提问能力评估 → 认知边界是不是伪命题 → 黑暗地图。起点是 A 类问题，终点是 C 类问题。地图没变大多少，但有两块区域，从微光变成了灯泡。
